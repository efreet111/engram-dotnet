# Adoption Guide

> **Referencia opcional** — esta guía describe la adopción incremental de FlowDoc. No es un proceso obligatorio para engram-dotnet.

> FlowDoc can be adopted incrementally based on your team's needs.

## Quick Start

1. Create HUs in `docs/tasks/HU-001-HU-099/`
2. Use templates from `docs/templates/`
3. Follow the SDD cycle
